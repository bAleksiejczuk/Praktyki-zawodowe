Application-for-creating-various-types-of-content is an api that allows you to add articles, events, procedures and display them.

Creating an Article
- The user enters the title and content of the article


Creating an Event

- name of the event
- description
- start and end time
- place



Creating procedures
- title
- description
- steps to a given procedure

Creating procedure steps
- step number
- description

Adding a comment for a specific article

Adding a user

User's declaration that they will take part in a given event

Displaying all of the above events
