AlkoLegro is an api that allows you to display products contained in the database and create users.

The product search engine has filters:
- Price from / to
- Categories
- Search for all articles starting with a given letter
- Search for all available products

Creating a user
- Checking if user's e-mail contains @
- The password provided by the user is hashed

Displaying orders
- Listing of all products ordered by the user
- Displaying the quantity of a given product ordered by the user
- Displaying the price that the user has to pay for the products
